# 🎬 Snap Studio

**Snap Studio** is an Android application built in **Java** for creating, editing, and sharing photos and videos with fun effects, filters, and stickers. The app provides a simple yet powerful interface for mobile content creation.

---

## 🚀 Features

- 📸 Capture photos and videos directly from the app  
- 🎨 Apply filters, stickers, and effects  
- ✂️ Basic editing: crop, rotate, adjust brightness/contrast  
- 💾 Save edited media to local storage  
- 📤 Share photos and videos on social media platforms  
- ⚡ Smooth and responsive UI  

---

## 🛠️ Tech Stack

- **Java** – Programming language  
- **Android Studio** – IDE for development  
- **Android SDK** – Native Android development tools  
- **Material Design** – UI components  
- **Media APIs** – Camera, storage, and editing features  

---

## 📂 Project Structure

```

app/src/main/java/com/photo/editor/snapstudio/
┣ activities/      # MainActivity, EditorActivity, etc.
┣ adapters/        # RecyclerView adapters
┣ models/          # Data models
┣ utils/           # Helper classes
┣ filters/         # Image and video filters
┗ SnapStudioApp.java # Application class
res/
┣ layout/          # XML layouts
┣ drawable/        # Images and icons
┣ values/          # Colors, strings, styles

````

## ⚙️ Getting Started

### Prerequisites

- Java JDK installed  
- Android Studio installed  
- Android device or emulator  

---

### 🔧 Installation

Clone the repository

```bash
git clone https://github.com/shubham230523/snap_studio.git

```

Open the project in Android Studio

Build and run the app on your device or emulator
